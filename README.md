## nostalgia-timeline

This repo contains the timeline app for [nostalgia](https://nostalgia-dev.github.io).

### Setup

    git clone https://github.com/nostalgia-dev/timeline
    pip install -r requirements.txt

### Running

    python timeline.py

### Updating the code

    git pull
